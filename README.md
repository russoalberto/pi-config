# pi-config

Personal [pi](https://github.com/earendil-works/pi-mono) coding agent configuration.

## Contents

- `settings.json` — Base settings (theme, provider, packages). Models excluded.
- `extensions/` — Custom TypeScript extensions (RTK, permission gate, review).
- `skills/` — Symlinks to installed skills.
- `themes/` — Custom themes (gruvbox-dark).
- `AGENTS.md` — Project-level agent instructions.

## Setup

```bash
git clone git@github.com:russoalberto/pi-config.git ~/.pi/agent
```

## Structure

```
~/.pi/agent/
├── AGENTS.md
├── extensions/
├── settings.json
├── skills/
└── themes/
```
